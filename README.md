# Family_Todo

An Flask API used to perform 
CRUD(Create, Read, Update, Delete) on:
- Family Objects
- Member Objects
- TodoItem Objects

## Tech Used

1. Flask
2. SQLAlchemy
3. Pyjwt
4. Python-dotenv

## Features

1. Family's have a list of Members and TodoItems.

2. Member's have their own Todo list to store their TodoItems.
